      # 1. npm

      # 1.1 npm安装

      • 1、先下载nodejs：https://nodejs.org/en/
      • 2、安装完毕后打开console输入以下命令查看nodejs是否安装成功
        PS G:\安装包> node -v
v12.18.3
      • 3、设置registry
        npm config set registry https://registry.npm.taobao.org
        此方法可用于解决以下问题：
        npm ERR! code EAI_AGAIN
npm ERR! errno EAI_AGAIN
npm ERR! request to https://registry.npmjs.org/@babel/helper-plugin-utils/-/helper-plugin-utils-7.10.4.tgz failed, reason: getaddrinfo EAI_AGAIN registry.npmjs.org
      • 4、设置代理为空
        npm config set proxy null
        此方法可用于解决以下问题：
        npm ERR! code EAI_AGAIN
npm ERR! errno EAI_AGAIN
npm ERR! request to https://registry.npm.taobao.org/@babel/plugin-syntax-nullish-coalescing-operator/download/@babel/plugin-syntax-nullish-coalescing-operator-7.8.3.tgz failed, reason: getaddrinfo EAI_AGAIN registry.npm.taobao.org

      # 2. yarn

      使用npm安装yarn：

      npm install -g yarn

      查看yarn版本:

      yarn --version

      如果使用PowerShell出现以下问题，那么可以使用cmd来执行相关命令：

      yarn : 无法加载文件 C:\Users\Administrator\AppData\Roaming\npm\yarn.ps1，因为在此系统上禁止运行脚本。

      # 3. window10使用vuepress

      # 3.1 项目搭建

      需要先准备好npm、yarn环境，准备好后接下来开始搭建项目。

      • 3.1.1 先创建一个目录
      md ryansBlog
cd ryansBlog
      • 3.1.2 初始化项目, 需要填写一些项目信息，不需要配置的就直接回车，最后选yes即可。回车后会在当前目录下生成package.json文件
      npm init
      • 3.1.3 安装vuepress依赖
      npm install -D vuepress

      • 3.1.4 环境准备好了，可以创建第一篇文章啦，在ryansBlog下创建一个目录，如java。在java下创建一个README.md，这个将是网站的首页，可以现在里面写入随便一点内容。

      • 3.1.5 修改package.json，添加两行脚本（dev和build那两行）：

      "scripts": {
  "test": "echo \"Error: no test specified\" && exit 1",
  "dev": "vuepress dev myblog",
  "build": "vuepress build myblog"
},

      -- 3.1.6 启动项目，启动成功后浏览器直接访问http://localhost:8080/myblog/即可。

      npm run dev

...(此处省略加载信息)，启动成功后的信息：
success [22:05:13] Build b311df finished in 21450 ms!
> VuePress dev server listening at http://localhost:8080/myblog/
